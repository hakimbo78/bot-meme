CONCLUSION
================================================================================

DexScreener FREE API has SEVERE limitations:
1. No endpoint to get ALL pairs for a chain
2. /search endpoint returns max 30 results
3. Results are FILTERED by relevance to query
4. No 'latest' or 'new' endpoint available

IMPLICATION:
- Keyword strategy will MISS random-named coins
- We can only detect coins matching our keywords
- This is a FUNDAMENTAL limitation of the FREE API

ALTERNATIVES:
1. DexScreener Pro API (paid) - has more endpoints
2. Direct blockchain scanning (expensive, CU-heavy)
3. Use multiple data sources (Birdeye, DEXTools, etc.)
